Portfolio Website v2
Overview

I designed and developed the Portfolio v2 website, a minimalist showcase targeting potential clients. This project reflects my interest in clean, engaging designs, focusing on typography and subtle design choices to enhance user interaction. The goal was to create a space that blends my passion for timeless design and coding.

Client: Personal Project
Date: October 2024
Role: Front-End Developer, UI/UX Designer
Tools Used: HTML, CSS, JavaScript, Adobe XD, Bootstrap
project goals

The main objective of this project was to create a visually appealing website that showcases my work as an software engineer. I wanted to emphasize simplicity while keeping the layout functional and inviting.

Target Audience: Potential clients, graphic designers
Core Focus: Typography, minimalism, and a seamless user experience
Objective: A fully responsive and smooth user interface
process

01. Research & Inspiration
I began by exploring modern portfolio designs on Pinterest. The goal was to find inspiration that combined simplicity with rich typography to highlight the minimal aesthetic.

02. Wireframing & Prototyping
I created wireframes in Adobe XD, focusing on a clean layout with bold headlines, few images, and a lot of white space to maintain a modern look.

03. Development
Using HTML and CSS, I developed the layout and ensured it was responsive across all devices. Bootstrap helped with grid management, and JavaScript added some minor interactivity.

04. Testing & Feedback
I tested the site for functionality on multiple devices and browsers to ensure it was responsive. A few design tweaks were made based on feedback from fellow developers and designers

challenges & solutions

Challenge: Finding the exact font i used.
Solution: When i designed my website i used a particular font that i could not use on my actual website. Therefore i did my research and found a similar font to the one i used.


Challenge: Responsiveness
Solution: There's a trick that i use to make sure my website is responsive to all devices using media queries. and i found margins that actually work.
outcome

The final design achieved the clean, minimalist look I aimed for, with engaging typography and a smooth, user-friendly experience. It successfully targets designers and developers who appreciate simplicity and functionality in design.
